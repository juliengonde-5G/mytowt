1. How to fill in your Packing List

Your packing list is pre-filled with voyage and booking information from TOWT. You need to complete the yellow fields with your cargo details.

Each batch represents a shipment portion. You can add multiple batches if your cargo has different types of goods or different shippers.

Tip: Save your progress regularly by clicking the save button at the bottom of the page.

2. Required fields

The following fields are mandatory for each batch:

FieldDescription
Customer nameYour company name
ShipperFull name, address, postal code, city, country
Notify partyFull name, address, postal code, city, country
ConsigneeFull name, address, postal code, city, country
Type of goodsDescription of the cargo
Pallet quantityNumber of pallets in this batch
DimensionsLength, width, height in cm per pallet
WeightWeight in kg per pallet

3. Pallet types and dimensions

TOWT vessels accept the following pallet formats:

FormatDimensions
EPAL (Europe)120 x 80 cm
US Pallet120 x 100 cm
Port Pallet120 x 100 cm
IBC120 x 106 cm
Big Bag120 x 103 cm
Barrel 120x120123 x 123 cm
Barrel 140x140143 x 143 cm
Basket constraint: Maximum pallet size is 380 x 150 cm, height 2.2 m, max weight 5.1 tonnes. Oversized cargo will be positioned in dedicated zones.

4. Uploading documents

Use the Documentation tab to upload supporting documents (invoices, certificates, photos, etc.).

Accepted formats: PDF, Excel, Word, images. Maximum file size: 20 MB.

You can delete your own uploaded documents at any time. Documents uploaded by TOWT staff cannot be deleted from this portal.

5. Tracking your cargo position

Once your cargo is assigned to a zone in the vessel, you can see its position in the Position tab.

A vessel schematic shows the exact zone where your cargo is stowed. The vessel is divided into 18 zones across 3 decks.

6. Contact us

If you have any questions, use the Contact tab to send us a message directly.

You can also reach us at: operation@towt.eu

1. Comment remplir votre Packing List

Votre packing list est pre-remplie avec les informations de voyage et de reservation TOWT. Vous devez completer les champs avec les details de votre cargaison.

Chaque batch represente une portion de votre chargement. Vous pouvez ajouter plusieurs batches si votre cargaison contient differents types de marchandises ou differents expediteurs.

Conseil : Sauvegardez regulierement en cliquant sur le bouton de sauvegarde en bas de page.

2. Champs obligatoires

Les champs suivants sont obligatoires pour chaque batch :

ChampDescription
Nom du clientRaison sociale de votre entreprise
Expediteur (Shipper)Nom, adresse, code postal, ville, pays
NotifyNom, adresse, code postal, ville, pays
Destinataire (Consignee)Nom, adresse, code postal, ville, pays
Type de marchandiseDescription de la cargaison
Nombre de palettesQuantite de palettes dans ce batch
DimensionsLongueur, largeur, hauteur en cm par palette
PoidsPoids en kg par palette

3. Types de palettes et dimensions

Les navires TOWT acceptent les formats de palettes suivants :

FormatDimensions
Europalette (EPAL)120 x 80 cm
Palette US120 x 100 cm
Palette portuaire120 x 100 cm
IBC120 x 106 cm
Big Bag120 x 103 cm
Barrique 120x120123 x 123 cm
Barrique 140x140143 x 143 cm
Contrainte panier : Dimensions maximales par palette : 380 x 150 cm, hauteur 2,2 m, poids max 5,1 tonnes. Les colis hors-format seront positionnes dans des zones dediees.

4. Deposer des documents

Utilisez l'onglet Documentation pour deposer vos documents justificatifs (factures, certificats, photos, etc.).

Formats acceptes : PDF, Excel, Word, images. Taille maximale : 20 Mo.

Vous pouvez supprimer vos propres documents a tout moment. Les documents deposes par l'equipe TOWT ne peuvent pas etre supprimes depuis ce portail.

5. Suivre la position de votre marchandise

Une fois votre cargaison affectee a une zone du navire, vous pouvez voir sa position dans l'onglet Position.

Un schema du navire montre la zone exacte ou votre marchandise est arrimee. Le navire est divise en 18 zones sur 3 ponts.

6. Nous contacter

Si vous avez des questions, utilisez l'onglet Parlez avec nous pour nous envoyer un message directement.

Vous pouvez aussi nous joindre a : operation@towt.eu
